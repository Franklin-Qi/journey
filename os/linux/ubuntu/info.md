# info 命令

info 命令和 man 命令不同的是，info 命令的帮助信息是一套完整的资料，每个单独命令

的帮助信息只是这套完整资料中的某一个小章节。可以把 info 帮助信息看成linux独立的

电子书，所以每个命令的帮助信息都会和书籍一样，拥有章节编号。

## 快捷键
H 或者 h 查看帮助文档

d 可以查看Linux 命令目录，很好的文档目录，可以帮助理解linux命令之间的关系

C-n 下一行

C-p 上一行

n 下一章节

p 上一章节

/ - { }  / 向下查找，{ 上一个结果 } 下一个结果

enter 切换到指定地址

tab 切换到下一地址
